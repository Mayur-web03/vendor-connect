# 📄 Vendor Connect – Invoice Management Web App

## 📌 Overview
🚀 A dynamic web application that allows users to generate and manage invoices with real-time calculations using JavaScript.

## 🔗 Live Demo
https://mayur-web03.github.io/vendor-connect/

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (ES6)

## ✨ Features
- Dynamic invoice generation
- Real-time price calculations
- Form validation and input handling
- Responsive UI design
- DOM manipulation for live updates

## ⚙️ Setup

1. Clone the repository  
2. Open the project folder  
3. Run `index.html` in your browser  

## 📚 Learnings
- DOM manipulation and event handling  
- Form validation techniques  
- Building dynamic UI without frameworks  

## 🎯 Purpose
This project simulates a real-world invoice system where users can manage vendor transactions efficiently through a simple web interface.

## 🚀 Key Highlight
- Built complete invoice generation logic using vanilla JavaScript without any frontend framework
